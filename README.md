# generator-scrapy
> Yeoman generator for scrapy app.

## installation
```shell
git clone git@github.com:afeiship/generator-scrapy.git
cd generator-scrapy
npm i && npm link
```

## usage
```shell
yo @jswork/scrapy

# create model
yo @jswork/scrapy:model nx_page
yo @jswork/scrapy:model nx_url
```
